Download dicomParser Library

If you're having issues with the CDN, download the library file locally:

  1. Click this link: Download dicomParser.min.js
  2. Save the file in the same folder as index.html
  3. Update index.html line 87 to use: <script src="dicomParser.min.js"></script>

Or use this direct link: https://unpkg.com/dicom-parser@1.8.21/dist/dicomParser.min.js
